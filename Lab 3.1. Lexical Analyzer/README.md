# diffComparison — Comparador de similitud de código Python

Detecta similitud entre programas Python usando cuatro estrategias: texto plano con difflib, tokens normalizados con difflib, suffix array con texto plano y suffix array con tokens.

## Requisitos

```bash
pip install matplotlib numpy
```

> `difflib`, `tokenize`, `io`, `keyword` y `os` son parte de la biblioteca estándar de Python, no requieren instalación.

## Uso

Abre y ejecuta todas las celdas del notebook:

```bash
jupyter notebook diffComparison.ipynb
```

O en VSCodium: abre el archivo y presiona **Run All**.
